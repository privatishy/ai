# Автоматизация процесса оплаты питания

Система автоматического распознавания блюд для школьной столовой

---

## Цель проекта

Разработка и внедрение работающей нейросетевой модели для автоматического распознавания блюд на подносе и расчете стоимости в школьной столовой с точностью не менее 90% и временем обработки изображения не более 2 секунд.

---

## Быстрый старт

### Требования

- Python 3.10+
- Веб-камера или IP-камера
- 4+ GB RAM
- 10+ GB свободного места на диске

### Установка

1. Клонируйте репозиторий:
   git clone <url-репозитория>
   cd ai

2. Создайте виртуальное окружение:
   python -m venv venv
   venv\Scripts\activate

3. Установите зависимости:
   pip install -r requirements.txt

4. Запустите систему:
   python main.py

5. Откройте веб-интерфейс:
   http://localhost:5000

## Конфигурация

### Переменные окружения (.env)

MODEL_PATH - Путь к модели (weights/yolov8m.pt)
MODEL_CONFIDENCE - Порог уверенности (0.5)
CAMERA_ID - ID камеры (0)
FLASK_PORT - Порт сервера (5000)
USE_CUDA - Использовать GPU (False)

### Классы для распознавания

1. borsch (борщ) - 85 руб.
2. pasta (макароны) - 65 руб.
3. cutlet (котлета) - 75 руб.
4. salad (салат) - 45 руб.
5. tea (чай) - 25 руб.
6. compote (компот) - 30 руб.
7. bread (хлеб) - 15 руб.

## Устранение неполадок

### Частые проблемы

ModuleNotFoundError - Выполните: pip install -r requirements.txt
Camera not found - Проверьте CAMERA_ID в .env
Low FPS - Нормально для CPU-режима (4-5 FPS)
Model not loading - Проверьте путь в MODEL_PATH

### Диагностика

Проверка Python: python --version
Проверка пакетов: pip list
Проверка GPU: python -c "import torch; print(torch.cuda.is_available())"
Проверка камеры: python -c "import cv2; cap = cv2.VideoCapture(0); print(cap.isOpened())"
