# Система автоматизации оплаты питания

Система автоматического распознавания блюд для школьной столовой

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?style=flat-square&logo=github)](https://github.com/privatishy/ai)

## О проекте

School Canteen AI — работающая система автоматизации оплаты питания в школьной столовой. Проект использует нейросетевую модель YOLOv8m для распознавания блюд на подносе в реальном времени.

Проект разработан для ЧОУ «Газпром школа Санкт-Петербург» в рамках индивидуального проекта 2025-2026.

**Автор:** Кулик Арсений Олегович, 10 класс «Е»  
**Научный руководитель:** Деев Андрей Павлович

## Установка

**Требования:**

- Python 3.10 или выше
- Веб-камера или IP-камера
- 4+ GB RAM
- 10+ GB свободного места

**Шаги:**

1. Клонирование репозитория:
   git clone https://github.com/privatishy/ai.git
   cd ai

2. Создание виртуального окружения:
   python -m venv venv

3. Активация окружения:

   # Windows

   venv\Scripts\activate

   # macOS/Linux

   source venv/bin/activate

4. Установка зависимостей:
   pip install -r requirements.txt

5. Запуск системы:
   python main.py

6. Открытие веб-интерфейса:
   http://localhost:5000

## Использование

**Главная страница (http://localhost:5000):**

## API Endpoints

| Endpoint         | Метод | Описание                     |
| ---------------- | ----- | ---------------------------- |
| /                | GET   | Главная страница             |
| /admin           | GET   | Админ-панель                 |
| /video_feed      | GET   | MJPEG поток видео            |
| /api/detect      | GET   | JSON с данными распознавания |
| /api/export-logs | GET   | Экспорт логов (ZIP)          |

## Конфигурация

**Переменные окружения (.env):**

MODEL_PATH=weights/yolov8m.pt
MODEL_CONFIDENCE=0.5
CAMERA_ID=0
FLASK_PORT=5000
USE_CUDA=False

## Диагностика проблем

| Проблема            | Решение                         |
| ------------------- | ------------------------------- |
| ModuleNotFoundError | pip install -r requirements.txt |
| Camera not found    | Проверьте CAMERA_ID в .env      |
| Low FPS             | Нормально для CPU-режима        |
| Model not loading   | Проверьте путь в MODEL_PATH     |
