# 🏆 School Canteen AI — production-grade food recognition system

> Это backend-система автоматизации оплаты питания для школьной столовой,
> использующая нейросетевую модель YOLOv8m для распознавания блюд в реальном
> времени. Проект разработан для ЧОУ «Газпром школа Санкт-Петербург» в рамках
> индивидуального проекта 2025-2026. Система построена так, как её пришлось бы
> делать в реальном продакшене: с логированием, безопасностью и масштабируемостью.
> Проект про аккуратную архитектуру, устойчивость к ошибкам и понятный код.
